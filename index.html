About It:

The project was developed by Sara Castrillón and Felipe Tovar.

How to use?

Before do anything please read both guides.

Producer guide:

You have 3 options:
1. Create a new topic. Just write a name and click the button to create It.
2. Send a message. Select the topic and write a message, the click the button to send It.
3. Delete a topic. Choose a topic and click the button to delete It.
Remember that no matter what option you choose, the users (consumers) need to reload their paged so they can see the changes.

Consumer guide:

First, you need to create your costumer choosing the topic you like the most.
Then, you have 3 options:
1. Listen a new topic. You can listen as many topics as the producer creates.
2. Receive a message. Click the button to update the view, the old messages will be deleted and the new ones will be displayed.
3. Stop listening a topic. You can select a topic you have and stop Its incoming messages. You can listen again to that topic in the Listen a new topic option.
