Producer zone:

Create a new topic:

This option will create a new topic for all the users.
Note that this application only works with pull services so if you create a topic, the consumers need to reaload their pages to update their available topics.

Send a message:

This option will send a message for all the users that listen the topic you select.
Note that this application only works with pull services so if you send a message, the consumers need to reaload their pages to update their messages.

Delete a topic:

This option will delete a topic for all the users.
Note that this application only works with pull services so if you delete a topic, the consumers need to reaload their pages to update their available topics.
